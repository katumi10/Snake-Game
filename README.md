# JavaScript Snake Game

## Overview
This is a classic Snake game built with HTML, CSS, and JavaScript. The player controls a snake, and the objective is to eat food, making the snake longer while avoiding collisions with the walls and itself.

## Features
- Move the snake with arrow keys
- Eat food to grow the snake
- The game ends when the snake collides with the wall or itself
- Displays score based on the snake's length

## How to Play
1. Open `index.html` in your browser.
2. Use the **arrow keys** to control the snake's movement:
   - **Up**: Move up
   - **Down**: Move down
   - **Left**: Move left
   - **Right**: Move right
3. Eat the food (red square) to grow the snake.
4. Avoid hitting the walls or your own tail. If you do, the game is over.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/katumi10/snake-game.git
   ```
2. Open `index.html` in your browser to start the game.
